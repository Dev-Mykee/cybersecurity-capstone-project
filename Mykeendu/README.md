## Methodology (CRISP-DM)
This project follows the **CRISP-DM** (Cross-Industry Standard Process for Data Mining) framework:
1. **Business/Data Understanding** — Understand the cybersecurity domain and dataset
2. **Data Preparation** — Clean, transform, and engineer features
3. **Exploratory Data Analysis** — Visualize patterns and anomalies
4. **Modeling** (Optional) — Train and evaluate machine learning models
5. **Evaluation & Deployment** — Interpret results and draw conclusions
